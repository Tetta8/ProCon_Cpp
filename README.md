## フォルダ構成
 - learn    :参考図書の演習問題として実施したソースコードを格納
   - 章ごとにフォルダを作成する。
 - challenge:参考図書無しで調整したソースコードを格納
   - 問題セットごとにフォルダを作成する。

## 命名規則
 - main.cpp:メイン関数を記載するソースコード
 - main.h:メインのヘッダファイル
 - その他：クラス名をファイル名とする。
